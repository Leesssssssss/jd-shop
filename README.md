# jd-shop
vue写的仿京东商城

前端页面使用vue.js框架</br>
后端使用egg.js传递数据</br>
服务器语言使用node.js</br>
数据库使用Mongo存储用户信息</br></br>

目前功能不是很完善，还在陆续更新</br></br>

完成的界面：</br>
1.首页</br>
2.用户登录</br>
3.用户注册</br>
4.分类界面</br>
5.个人信息</br>
6.商品详情</br>
7.购物车</br>
8.账号管理</br>
9.收货地址管理</br>
10.确认订单</br>
11.订单支付</br></br>
